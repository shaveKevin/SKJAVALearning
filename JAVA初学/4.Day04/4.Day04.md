# Java 对象和类
Java作为一种面向对象的语言，支持以下基本概念。
* 多态
* 继承
* 封装
* 抽象
* 类
* 对象
* 实例
* 方法
* 重载
这里主要研究的是对象和类的概念。
* **对象：** 对象是一个类的实例，有状态和行为。例如：一条狗是一个对象，它的状态有颜色，名字，品种等；行为有摇尾巴，叫，吃等。
* **类：** 类是一个模板，它描述了一类对象的行为和状态。
